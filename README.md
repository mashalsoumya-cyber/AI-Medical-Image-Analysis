# AI-Powered Medical Image Analysis System

## Overview
This project is an AI-based medical image analysis system that classifies chest X-ray images as **Normal** or **Pneumonia**.

## Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib
- Scikit-learn
- Flask

## Dataset
Chest X-ray dataset with:
- train
- val
- test

Classes:
- NORMAL
- PNEUMONIA
